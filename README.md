# Side Quest Week 3: Full Fighting Game

## Setup and Interaction Instructions

A two-player fighting game where each player can move, attack, and block using separate keyboard controls.
Defeat your opponent through attacking, positioning, and defense, then press ENTER to start a new match or rematch.

**Player 1 Controls:**

- Move: A / D
- Attack: F
- Block: G

**Player 2 Controls:**

- Move: Arrow Keys
- Attack: K
- Block: L

Press **ENTER** to start or rematch.
